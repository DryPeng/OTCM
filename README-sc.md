# OTCM - 千首华曲
一千首歌曲，不仅包含普通话，还有粤语、闽南语等。各种年代/歌手的歌曲，目前来看似乎情歌较多。

## 整理收录版本规则
- 无政治倾向（可讽刺）
- 无抄袭
- 中文歌曲
   - 包括但不限于，官话、晋语、平话、吴语、徽语、赣语、湘语、闽东语、闽北语、邵将语、闽南语、琼雷话、闽中语、莆仙语、客家话、瓦乡话等


一般来说千首华曲都是按照这个标准收集的，有疑问可以随时使用issues。我们也希望大家可以帮助我们一起收集。
